# CSS Inserttags to include (resized) images in stylesheets
Insert tags for style sheets

### Install:


--
### Example:
For resized image as an asset
```
.class {
    background: url({{) #fff;
}
```

For an embedded image
```
.class {
    background: url() #fff;
}
```
